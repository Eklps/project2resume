## 与 v1.0 对比

| 维度 | v1.0 | v2.0 |
|------|------|------|
| 输出格式 | Markdown | **HTML**（可直接打印） |
| 覆盖范围 | 仅项目经历 | **完整简历**（个人信息+教育+项目+技能） |
| 个人信息 | 占位符 | **自动采集并持久化** |
| 项目数量 | 单项目 | **支持多项目** |
| 专业技能 | 无/手动 | **自动汇总 + 用户补充** |
| 排版效果 | 依赖渲染器 | **A4 精确排版，可打印** |
| 交互功能 | 无 | **联系方式一键复制** |
