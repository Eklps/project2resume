## 岗位关键词覆盖表

### 后端开发岗

必须覆盖（至少 5 项）：
```
架构设计, RESTful API, 数据库设计, 缓存策略, 并发控制,
事务管理, 消息队列, 安全认证, 性能优化, 微服务
```

### 前端开发岗

必须覆盖（至少 5 项）：
```
组件化开发, 状态管理, 路由设计, 性能优化, 响应式布局,
跨端适配, 打包优化, 前端工程化, UI/UX, 前后端联调
```

### 全栈开发岗

必须覆盖（前后端各至少 3 项）：
```
后端: 架构设计, API 设计, 数据库
前端: 组件化, 状态管理, 性能优化
通用: 工程化, DevOps, 系统集成
```

### Java 后端岗（细分）

高频面试关键词：
```
Spring Boot, MyBatis, Redis, MySQL, RabbitMQ/Kafka,
分布式锁, 事务, AOP, 线程池, JVM 调优,
设计模式, RESTful, 微服务, Docker
```
